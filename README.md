# Сайт адвоката Соколова А.И. — Шаблон №1

Статический сайт для адвоката. Три страницы, тёмная тема, золотой акцент. Без фреймворков — чистый HTML/CSS/JS.

**Живой сайт:** задеплоен на Vercel (lawyer-sokolov.vercel.app)  
**Репозиторий:** github.com/IvanSeliverovOOOVozdux/lawyer-sokolov

## Как вносить изменения

### Текст и контакты
Открой нужный `.html` файл в любом редакторе (VS Code, Notepad++) и найди нужный текст через Ctrl+F.

Основные места для правки в `index.html`:
- Имя и должность — ищи `Соколов А.И.`
- Телефон — ищи `+7 (495) 123-45-67`
- Адрес — ищи `Пречистенская набережная`
- Статистика — ищи `12+`, `340+`, `89%`

### Цвета
В начале каждого файла есть блок `:root { ... }` — все цвета там:
```css
--gold: #c09a3f;   /* золотой акцент */
--bg:   #0b0f19;   /* основной фон */
--text: #e8e2d4;   /* основной текст */
```
Поменяй `#c09a3f` на любой другой цвет — изменится везде сразу.

## Как задеплоить изменения

После правки файлов — три команды в терминале:

```bash
cd C:\Users\sasha\lawyer-sokolov
git add .
git commit -m "описание изменений"

```
